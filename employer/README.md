# JobAI - Employer Application

## Setup Instructions

### Backend (Port 8002)
```bash
cd backend
composer install
php artisan serve --port=8002
```

### Frontend (Port 3002)
```bash
cd frontend
npm install
npm run dev
```

## Configuration
- Backend API: http://localhost:8002
- Frontend: http://localhost:3002
- Database: ai-job-recommendation_db (shared)
