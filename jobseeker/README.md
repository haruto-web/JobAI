# JobAI - Jobseeker Application

## Setup Instructions

### Backend (Port 8001)
```bash
cd backend
composer install
php artisan serve --port=8001
```

### Frontend (Port 3001)
```bash
cd frontend
npm install
npm run dev
```

## Configuration
- Backend API: http://localhost:8001
- Frontend: http://localhost:3001
- Database: ai-job-recommendation_db (shared)
