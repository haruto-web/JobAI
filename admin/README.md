# JobAI - Admin Application

## Setup Instructions

### Backend (Port 8003)
```bash
cd backend
composer install
php artisan serve --port=8003
```

### Frontend (Port 3003)
```bash
cd frontend
npm install
npm run dev
```

## Configuration
- Backend API: http://localhost:8003
- Frontend: http://localhost:3003
- Database: ai-job-recommendation_db (shared)
